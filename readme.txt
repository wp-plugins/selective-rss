== Description ==

This is a simple Plugin that allows you to embed RSS feed items into Pages or Posts. It also optionally allows you to choose how many items to display and allows you to limit items to ones that contain certain words in the titles.

== Installation ==

1. Upload the files to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Place [srss ...] tag into a post or page.

Examples
Simple: [srss url=http://www.domain.example/feed]
Complex: [srss url=http://www.domain.example/feed,limit=10,filter=word1;word2]
